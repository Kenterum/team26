Hour of Code Project

Hour of Code project helps school students to learn algorithms and create their code and with funny game characters. First of all, let us tell you about "Hour of Code" briefly, in the project students had to write the algorithms for the games in different forms and pass the levels. Students also can see in the videos how they can work the new blocks. In this project, we spent two magnificent hours with the students of Aghdam (Karabakh) Secondary School No.32. For acquaintance with the project first, we showed them a Presentation generally about the meaning of the algorithms and their real-life utilization. By our instructions, they completed the courses in two days and received their certificates. The main aim of the project was the contribution of computer knowledge to the interested students.
